切换pem为账号登陆

  1. 下载xxx.pem文件到本地

  2. 命令行cd到xxx.pem所在路径，使用pem登陆

    angelscript

  3. 创建root密码

    ebnf

输入两次密码，期间没有任何提示

  1. 切换到root身份

    ebnf

  2. 使用root身份编辑亚马逊主机的ssh登录方式

    awk

把 PasswordAuthentication no 改为 PasswordAuthentication yes
如果没有则新插进去

  1. 重启sshd使修改生效

    ebnf

  2. 使用用户名密码远程连接

    nginx

搭建ftp服务器

  1. 使用 yum 安装 vsftpd

    cmake

  2. 启动 VSFTPD
    安装完成后，启动 FTP 服务：

    crmsh

启动后，可以看到系统已经监听了 21 端口

1c

  1. 配置 FTP 权限

编辑 /etc/vsftpd/vsftpd.conf，找到下面两处配置并修改：
禁用匿名用户 12 YES 改为NO

ini

禁止切换根目录 101 行 删除#

ini

  1. 重新启动 FTP 服务

    ebnf

  2. 创建 FTP 用户
    创建一个用户:web

    ebnf

  3. 为用户 ftpuser 设置密码

    jboss-cli

  4. 限制该用户仅能通过 FTP 访问
    限制用户 ftpuser只能通过 FTP 访问服务器，而不能直接登录服务器：

    awk

  5. 为用户分配主目录
    为用户 ftpuser创建主目录并约定：
    /data/ftp 为主目录, 该目录不可上传文件
    /data/ftp/pub 文件只能上传到该目录下

    haskell

  6. 设置访问权限

    haskell

  7. 设置为用户的主目录：

    haskell

  8. 后续工作
    注意：记得关闭防火墙或者开放FTP默认端口(21)

  • 关闭SELinux服务

    angelscript

  • 关闭防火墙

    ebnf
